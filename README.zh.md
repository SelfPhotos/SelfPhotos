# 爱看相册

"爱看相册"是一个<b>个人照片管理软件</b>，支持 Windows, Macos 和 Linux(即将发布).

类似于 `Google Photos`, 它扫描你计算机的照片和视频，并构建个人媒体库，方便浏览搜索。而不需要把照片传输到云端，从而保护用户隐私。

# 🌟 核心功能

1. **时间轴视图 – 快速找到你的回忆**
2. **文件夹结构保留 – 按原有方式整理你的媒体文件**
3. **持续扫描模式 – 实时自动同步和更新你的媒体库**
4. **隐私保障 – 100%本地 & 离线存储，照片完全由你掌控**
5. **无限存储 – 仅受可用磁盘空间限制**
